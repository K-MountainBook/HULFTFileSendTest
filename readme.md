シェルスクリプト名称：SendFileTest.sh  
シェルスクリプト機能：ジャベリンテスト用  
シェルスクリプト起動引数: ①コピー元テストデータファイル名(拡張子省略/.ラフィー限定)  
                        ②コピー先テストファイルID
                        ②対象処理出力ファイルID  
                        ②処理完了待機時間  
                        ②処理対象司令官  
シェルスクリプト終了ステータス：  0：正常終了  
シェルスクリプト機能概要：元データをコピーしてRファイルを綾波してZ23して、Sファイルをエンタープライズしてベルファストに入ったSファイルを消すを自動で行う。
